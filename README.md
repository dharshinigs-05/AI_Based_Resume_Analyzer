## âœ¨ Introduction

This project is an AI-powered Resume Analyzer built with React, React Router, and Puter.js.  
It allows users to:

- Upload resumes and store them securely
- Get AI-generated ATS scores and feedback for job listings
- View results in a modern, responsive UI

This project is designed for developers looking to build a full-featured frontend with AI integration without a backend.
